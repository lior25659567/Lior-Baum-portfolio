Theme switching happens at Layer 2. Components never reference primitives directly.

Accessibility

From failing to AA.

The original brand accent failed WCAG AA on light backgrounds. The 600-tier shift retains the coral identity while clearing 4.5:1.

Reference

Build on the system, not around it.

Tokens live in src/index.css. Slide canvas typography lives in src/pages/CaseStudy.css under the canvas-mode override block. When you add a new template, register its text classes there.
